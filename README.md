
# qb-signrobbery
Sign Robbery for QBCore

# Please wait till release to use. 

# Installation
* Download ZIP
* Drag and drop resource into your resources folder.
* Start resource
* Add Images into your inventory script
* Add Items into QBCore Shared lua
